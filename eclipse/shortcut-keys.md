Eclipse Shortcut Keys
===========================

 | Shortcut Key     | Function                 |
 |------------------|--------------------------|
 | Alt + Shift + Q  | Show Views               |
 | Ctrl + F7        | Circle Views             |
 | F12              | Activate Editor          |
 | Ctrl + PgUp      | Previous Editor Tab      |
 | Ctrl + PgDn      | Next Editor Tab          |
